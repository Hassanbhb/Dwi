# Dwi

Dwi is a simple application where you can post for others to see.

## Getting started

Make sure you have nodejs/npm already installed locally.

```terminal
git Clone https://github.com/Hassanbhb/Dwi.git Dwi
cd Dwi
npm install
```

### Running devlopment mode

```javasjcript
npm run devStart
```
